# Shared Layer
Common layer with reused modules without binding to business logic.

```bash
src/
└── shared/
      ├── ui/
      ├── assets/
      ├── types/
      ├── const/
      ├── config/
      ├── lib/
      └── api/
```

_**shared/ui**_
- general UIX components

_**shared/assets**_
- vendors files (i.e icons)

**_shared/lib_**
- general modules and hooks

_**shared/types**_
- types declaration

**_shared/config_**
- data that can be used in the code of the overlying layers

**_shared/api_**
- API for backend
